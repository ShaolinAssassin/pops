* *How to use it :* 

1. Copy/past all lines into a CHEATS.TXT file.
2. Put that file into the game VMC folder. 
3. Use uLE Text editor to active the wanted combinaison (+ $) and let the other lines unedited.


Ex : modes 3+4 enabled
