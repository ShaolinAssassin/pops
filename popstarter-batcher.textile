*How to use BATCHER_0.3.EXE ?*

1. Drop BATCHER_0.3.EXE where your .VCD are (recommended : either from CUE2POPS folder, or from your POPS folder) ;
2. Drop POPSTARTER.ELF in the same folder ;
3. Run BATCHER_0.3.EXE ;
4. Choose option :

* *A/* will create HDD launchers (no prefix .ELF) and VMC game folders, matching the .VCD files.
* *B/* will create USB launchers (XX. prefix .ELF) and VMC game folders, matching the .VCD files.
* *C/* will create SMB launchers (SB. prefix .ELF) and VMC game folders, matching the .VCD files.

If the VMC game folder already exists, it's not overwritten.

* *D/* will update all game launchers - same mode(s) as before the update.
* *E/* exits the app.
