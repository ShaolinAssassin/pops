*How to use BATCHER_0.3.1.EXE ?*

1. Drop BATCHER_0.3.1.EXE where your .VCD are (recommended : either in your CUE2POPS folder, or in your POPS folder for USB and SMB modes) ;
2. Drop POPSTARTER.ELF in the same folder ;
3. Run BATCHER_0.3.1.EXE ;
4. Choose option :

Options 1, 2 and 3 are useful when you want to *create all launchers for your games at once*. These will create HDD or USB or SMB launchers (no prefix .ELFs or XX. prefix .ELFs or SB. prefix .ELFs) and VMC game folders, matching the .VCD files. If the VMC game folder already exists, it's not overwritten.

Option 4 is useful when a new POPSTARTER version is out. It will *update all game launchers at once* - same mode(s) as before the update.

Option 5 exits the app.
