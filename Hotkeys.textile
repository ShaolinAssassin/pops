h1. *POPStarter Hotkeys*


______________________________________________________________________________________________________________


| *Hotkey* | *Description* | *Notes* |
| Select+L1+R1 | Native In-Game Reset | Can be changed using the IGR Behaviour Modifiers. |
| Select+L1+R2 | Enable the smooth texture mapping |  |
| Select+L2+R1 | Disable the smooth texture mapping |  |
| Select+L1+L2 | Turns the "scanlines generator":https://bitbucket.org/ShaolinAssassin/popstarter-documentation-stuff/wiki/scan_lines_preview OFF | Starts with OBT 14 |
| Select+R1+R2 | Turns the "scanlines generator":https://bitbucket.org/ShaolinAssassin/popstarter-documentation-stuff/wiki/scan_lines_preview ON | Starts with OBT 14 |
| Select+L2+R1 | Disable the smooth texture mapping |  |
| Select+L2+R2+Triangle | Opens the PS1 CD lid | If used with a BIOS.BIN, opening the lid at startup makes the MC manager and the CD Player reachable.  |
| Select+L2+R2+Up  | To Insert Disc 1 (DISCS.TXT line 1) | Part of the change disc feature. |
| Select+L2+R2+Right | To Insert Disc 2 (DISCS.TXT line 2) | Part of the change disc feature. |
| Select+L2+R2+Down | To Insert Disc 3 (DISCS.TXT line 3) | Part of the change disc feature. |
| Select+L2+R2+Left | To Insert Disc 4 (DISCS.TXT line 4) | Part of the change disc feature. |
| Select+L2+R2+Square | Close the PS1 CD lid | Closing the lid in the OSD will run the game. Part of the change disc feature. |
