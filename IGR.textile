h2. _IGR Behaviour Modifiers :_ 

POPStarter has his own IGR ( !https://bitbucket.org/repo/Mpy7Eg/images/2984521000-Playstation-Button-Select.png(title)! + !https://bitbucket.org/repo/Mpy7Eg/images/3507486238-Playstation-Button-R1.png(title)! + 
!https://bitbucket.org/repo/Mpy7Eg/images/1415872802-Playstation-Button-L1.png(title)! ). You can change the button combo using one PATCH/TROJAN ("IGR Behaviour Modifiers archive":https://bitbucket.org/ShaolinAssassin/popstarter-documentation-stuff/downloads/IGR%20Behaviour%20Modifiers.zip).

| *File name* | *Button combo* |
| TROJAN_0.BIN | Hold L1+L2+R1+R2+X+Down to open the IGR menu |
| TROJAN_1.BIN | Hold Start+Select to open the IGR menu |
| TROJAN_2.BIN | Hold L1+L2+R1+R2+Start+Select to open the IGR menu |
| TROJAN_3.BIN | Hold L1+L2+R1+R2+X+Down to terminate POPS (no IGR menu) |
| TROJAN_4.BIN | Hold Start+Select to terminate POPS (no IGR menu) |
| TROJAN_5.BIN | Hold L1+L2+R1+R2+Start+Select to terminate POPS (no IGR menu) |
| PATCH_0.BIN | Disables the IGR menu |

* *How to install :*
Copy one of the PATCH/TROJAN files to the VMC directory of your game, or to the POPS directory.
When copied to a VMC directory, it's used on the relative game only.
When copied to the POPS directory, it's used on all the installed games.

* *What to do if you don't want to overwrite your existing TROJAN_#.BIN/PATCH_#.BIN file :* 
# Change the number in the name of the file you want to copy;
# Open the file you want to copy with a hexadecimal editor, change the number in its header according to the one of the new file name;
# And finally copy the edited file.
POPStarter will refuse to load the PATCH/TROJAN file if the number in its filename doesn't match the number in its header.

h2. _IGR Textures:_ 

POPStarter has an IGR loader texture built-in so you can change the japanese IGR screen to one of your language. Just drop the "IGR_BG.TM2", "IGR_NO.TM2" and "IGR_YES.TM2" files into your POPS folder (hdd0:/__common /POPS, or mass0:/POPS, or smb0:/YourSharedFolder/POPS).

*Note :* .TM2 extension MUST be UPPERCASE for internal HDD - or it won't work. 

Some IGR textures made by the guys @"ASSEMbler":http://assemblergames.com/l/threads/ps2-pops-stuff.45347/. Kudos to them. Credits go to their original authors.

h2. _"Exit to [name of app]" function :_ 

POPStarter has a built-in launcher added to the in-game reset function. It looks for mc0:/BOOT/BOOT.ELF then mc1:/BOOT/BOOT.ELF. If not found/invalid, it exits to the PS2 browser. 
If you want to exit to some app (uLE_kHn for example), name it BOOT.ELF and place it in the BOOT folder of your memory card.
