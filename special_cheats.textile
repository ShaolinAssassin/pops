h1. *Special Cheats - Automated Codes*


______________________________________________________________________________________________________________

Some useful codes have been automated. Here are them :

| *Cheat Code* | *Description* | *Notes* |
| $SAFEMODE | Disable the cheat engine and only activate it after POPS has left the PS OSD | Some game codes do patch the memory area where the PS OSD is loaded, causing crashes and other garbage... |
| $CACHE1 | Makes POPS buffer 1 sector instead of 16 | |
| $COMPATIBILITY_0x## | Activate a compatibility mode | (where ## is a hexadecimal value) You can write as many $COMPATIBILITY_0x## as you want |
| $FAKELC | Loads a null LibCrypt magic word into the cop0 register | May be needed by some discs that have a messed up LC protection |
| $USBDELAY_# | Sets up the PFS wrapper USB delay | (# is a number) |
| $SMOOTH | Enable the "smooth texture":https://bitbucket.org/ShaolinAssassin/popstarter-documentation-stuff/wiki/SMOOTH mapping at startup | $S0003390 00000001 |
| $NOPAL | Disable POPStarters' PAL patcher | PATCH_9.BIN now obsolete |
| $FORCEPAL | Force the activation of the PAL patcher (POPS will run it PAL) and patches the BIOS region code to Euro (shows the boot screen in PAL)  | Useful for PAL VCDs that don't have a valid license text in their bootsector. PATCH_8.BIN now obsolete |
| $480p | Force 480p | Not compatible with $XPOS_####, $YPOS_##, $DWSTRETCH_####, $DWCROP_#### & NOT reliable ATM. See notes "here":http://assemblergames.com/l/threads/ps2-pops-stuff.45347/page-93#post-908556  |
| $XPOS_#### | Help to center the screen (horizontally) | # is a number (decimal). Work only in PAL and NTSC modes. |
| $YPOS_## | Help to center the screen (vertically) | # is a number (decimal). Work only in PAL and NTSC modes. |
| $DWSTRETCH_#### |  Stretches the display horizontally to your screen. | # is a number (decimal). Work only in PAL and NTSC modes. |
| $DWCROP_#### | Reduces/expands the display area width. | # is a number (decimal). Work only in PAL and NTSC modes. |
| $00507028 00000001 | Rumble Always On (Pad 1) |  |
| $005070B8 00000001 | Rumble Always On (Pad 2) |  |
| $=0507034 00000073 | Stay on Digital Mode (POPS, Pad 1) | // If analog is on |
| $00507034 00000041  |  | // Force digital mode |
| $=05070C4 00000073 | Stay on Digital Mode (POPS, Pad 2) | // If analog is on |
| $005070C4 00000041  |  | // Force digital mode |
| (code below)  | Left Stick is the D-Pad | Enables analogic joystick support for games that doesnt support it natively. |
